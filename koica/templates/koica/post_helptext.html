Utilisation du html

Le html peut être utilisé pour poster, les tags autorisés sont:

p i strong b ul ol li a h1 h2 h3 pre br img blockquote div
Les attributs autorisés sont:
href src width height lang class

A noter: les sauts de ligne sont automatiquement convertis en <br />

Code

Pour écrire du code il faut utiliser la balise <pre lang="">

Exemple:

<pre lang="python"> 

def foo():
	print "bar"
	return
</pre>

Cela s'affichera comme suit: {% pygment %} 
def foo():
    print "bar"
    return
{% endpygment %}
